# cimublog - v1
简单的四个模块：首页、知识、小工具、个人介绍、简单了支持响应式;
首页：待办、部分笔记和常用工具;
知识：存放个人笔记，面试题，快速入门和实用方法;
小工具：放置一些自己写的小工具，有用和没用的;
个人介绍：对个人的介绍，页面风格模仿Vue作者;
